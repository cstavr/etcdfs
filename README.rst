================================
etcdfs: Filesystem based on etcd
================================

A simple FUSE filesystem based on etcd.

Unsupported Operations
----------------------

* `chmod`
* `chown`
* `getxattr`
* `ioctl`
* `link`
* `listxattr`
* `readlink`
* `removexattr`
* `rename`
* `setxattr`
* `statfs`
* `symlink`
* `utimens`
